# Screenshots Folder

This folder contains screenshots of the BrotoRaise application for documentation purposes.

## Required Screenshots

Please add the following screenshots before final submission:

1. **student-dashboard.png** - Student dashboard view showing stat cards and complaints list
2. **admin-panel.png** - Admin panel with complaint management interface
3. **super-admin-dashboard.png** - Super admin view with admin management
4. **brobot.png** - BroBot AI assistant chat interface
5. **analytics.png** - Analytics dashboard with charts and filters
6. **mobile-view.png** - Mobile responsive design showcase

### Screenshot Guidelines
- Use high-quality PNG or JPG format
- Recommended resolution: 1920x1080 for desktop, 375x812 for mobile
- Include both light and dark theme views if applicable
- Ensure sensitive data is removed or anonymized
- Show the app in action with realistic demo data

### Tools for Screenshots
- Use browser DevTools for responsive mobile screenshots
- Consider tools like:
  - Screely (screely.com) for beautiful browser mockups
  - CleanShot X (Mac)
  - ShareX (Windows)
  - Browser built-in screenshot tools (F12 > Device Toolbar)

**Note:** Screenshots are referenced in the main README.md file.
